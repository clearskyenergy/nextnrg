Portfolio Rollup Backfill

One-time tool. Stamps the flat fields the Portfolio Command Center reads onto existing project docs, so sites created before the editor wiring stop showing $0 / Candidate Review.
Backfill logic per doc: • bessKwh — computed from the saved bessList (Σ kwh × qty). • stage — set to package only if the doc has no stage yet. • quoted — set true only if a real capex/revenue already exists on the doc. • capex / incentive / annualRevenue — only written if already present on the doc; never invented. A merge write is used, so nothing else in the site doc is touched.
